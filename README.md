## Overview

This repository contains the source code for my personal portfolio website. It represents my professional profile as a Computer Science and Engineering graduate, highlighting my technical skills, project work, and approach to building clean, reliable, and user-focused software solutions.

The portfolio is intentionally designed with a simple and structured layout to ensure clarity, ease of navigation, and a professional presentation. It serves as a single point of reference for recruiters and hiring teams to understand my capabilities and development mindset.

---

## Purpose

The primary objectives of this portfolio are to:

- Present my technical profile in a clear and professional format  
- Showcase relevant academic and personal projects  
- Demonstrate practical understanding of modern web technologies  
- Reflect readiness for entry-level software and IT roles  

---

## Technology Stack

- Frontend Framework: React  
- Language: TypeScript  
- Build Tool: Vite  
- Styling: Tailwind CSS  
- UI Components: shadcn/ui  

---

## Key Features

- Responsive and mobile-friendly design  
- Clean and minimal user interface  
- Structured project showcase  
- Dedicated sections for skills and background  
- Optimized for readability and performance
